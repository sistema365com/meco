Base HTML structure

This theme is a fluid layout with two columns and or three columns. All of the information within the page content area is nested within a div with id of "page-content". The sidebar left's (column #2) content is within a div with an id of "sidebar-left". The sidebar right's (column #3) content is within a div with an id of "sidebar-right". The general template structure is the same throughout the template. Here is the general structure.

HTML structure

<?php $this->beginPage() ?>
<!DOCTYPE html>
<!--[if IE 8]> <html lang="en" class="ie8"> <![endif]-->
<!--[if IE 9]> <html lang="en" class="ie9"> <![endif]-->
<!--[if !IE]><!--> <html lang="en"> <!--<![endif]-->

    <!-- START @HEAD -->
    <head>
        <!-- START @META SECTION -->
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
        <!--/ END META SECTION -->
      <title><?= $this->title ?></title>
        <?php $this->head();  ?>

    </head>
    <!--/ END HEAD -->

    <body class="page-session page-sound page-header-fixed page-sidebar-fixed">
    <?php $this->beginBody() ?>

        <!-- START @LOADING ANIMATION -->
        <div id="loading">
            <div class="loading-inner">
                <img src="<?= \Yii::$app->request->BaseUrl ?>/global/img/loader/flat/3.gif" alt="..."/>
            </div>
        </div><!-- /#loading -->
        <!--/ END LOADING ANIMATION -->

        <!-- START @WRAPPER -->
        <section id="wrapper">

            <!-- START @HEADER -->
            <?php echo $this->render("_header-admin") ?>
            <!--/ END HEADER -->

            <!-- START @SIDEBAR LEFT-->
            <?php echo $this->render("_sidebar-left") ;?>
            <!--/ END SIDEBAR LEFT -->

            <!-- START @PAGE CONTENT -->
            <?php echo $content ; ?>
            <!--/ END PAGE CONTENT -->

            <!-- START @SIDEBAR RIGHT -->
            <?php echo $this->render("_sidebar-right") ;?>
            <!--/ END SIDEBAR RIGHT -->

        </section><!-- /#wrapper -->
        <!--/ END WRAPPER -->

        <!-- START @BACK TOP -->
        <div id="back-top" class="animated pulse circle">
            <i class="fa fa-angle-up"></i>
        </div><!-- /#back-top -->
        <!--/ END BACK TOP -->


    <?php $this->endBody() ?>
    </body>
    <!--/ END BODY -->

</html>
<?php $this->endPage() ?>

Installation guide

This manual explains how to installation Blankon Yii2 on your project using Yii2 Framework.

Step Description
1

Download Yii2

Official site : http://www.yiiframework.com/download
2

Install yii2-basic application via composer

Detail installation with composer : http://www.yiiframework.com/doc-2.0/guide-start-installation.html
3

Copy merge assets, config, controllers, and module folder from Blankon-yii2 to your Yii2 folder.
4

Copy merge Blankon assets folder to your Yii2 project
5

Go to config folder, open web.php and change this line

'aliases'=>[
        '@asset'=>  'path assets Blankon'
    ]
6

Open browser and go to web folder, you will be redirect to admin module.

Example URL browser : http://localhost/blankon-fullpack-admin-theme/production/admin/yii2/web/index.php?r=admin%2Fdashboard%2Findex

Congratulations! you can now starting Blankon yii2 app :D
