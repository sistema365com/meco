Contains some samples to use c3js together with angularjs to give it a dynamic feel.

Official website : https://github.com/jettro/c3-angular-sample

SOURCE CODE (JS)

angular.module('blankonApp.charts.c3js', [])

        // =========================================================================
        // LINE
        // =========================================================================

            $scope.datapoints=[
                {'data1':30,'data2':50,'data3':130},
