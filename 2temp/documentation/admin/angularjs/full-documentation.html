Getting started

AngularJS lets you write client-side web applications as if you had a smarter browser. It lets you use good old HTML (or HAML, Jade and friends!) as your template language and lets you extend HTML’s syntax to express your application’s components clearly and succinctly. It automatically synchronizes data from your UI (view) with your JavaScript objects (model) through 2-way data binding. To help you structure your application better and make it easy to test, AngularJS teaches the browser how to do dependency injection and inversion of control.

Oh yeah and it helps with server-side communication, taming async callbacks with promises and deferreds. It also makes client-side navigation and deeplinking with hashbang urls or HTML5 pushState a piece of cake. The best of all: it makes development fun!

Base HTML structure

This theme is a fluid layout with two columns and or three columns. All of the information within the page content area is nested within a div with id of "page-content". The sidebar left's (column #2) content is within a div with an id of "sidebar-left". The sidebar right's (column #3) content is within a div with an id of "sidebar-right". The general template structure is the same throughout the template. Here is the general structure.

HTML structure

<!DOCTYPE html>
<html lang="en" data-ng-app="blankonApp">
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
        <title data-ng-bind="$state.current.data.pageTitle + ' | BLANKON AngularJS'"></title>
        <!-- CSS FILES-->
        <link data-ng-href="{{...}}" rel="stylesheet">
    </head>

    <body data-loading-animation>

        <!-- START WRAPPER -->
        <section id="wrapper">

            <!-- START HEADER -->
            <header data-ng-include="'partials/header.html'" id="header">
                ...
            </header><!-- /#header -->
            <!--/ END HEADER -->

            <!-- START SIDEBAR LEFT -->
            <aside data-sidebar-left-nicescroll data-sidebar-minimize data-ng-include="'partials/sidebar-left.html'" id="sidebar-left">
                ...
            </aside><!-- /#sidebar-left -->
            <!--/ END SIDEBAR LEFT -->

            <!-- START PAGE CONTENT -->
            <section id="page-content">

                    <!-- START HEADER CONTENT -->
                    <div class="header-content">
                            ...
                    </div><!-- /.header-content -->
                    <!--/ END HEADER CONTENT -->

                    <!-- START BODY CONTENT -->
                    <div data-ui-view class="body-content">
                            ...
                    </div><!-- /.body-content -->
                    <!--/ END BODY CONTENT -->

                    <!-- START FOOTER CONTENT -->
                    <div class="footer-content">
                            ...
                    </div><!-- /.footer-content -->
                    <!--/ END FOOTER CONTENT -->

            </section><!-- /#page-content -->
            <!--/ END PAGE CONTENT -->

            <!-- BEGIN SIDEBAR RIGHT -->
            <aside data-choose-themes data-navbar-color data-sidebar-color data-layout-setting data-ng-include=" 'partials/sidebar-right.html' " id="sidebar-right">
                ...
            </aside><!-- /#sidebar-right -->
            <!--/ END SIDEBAR RIGHT -->

        </section><!-- /#wrapper -->
        <!--/ END WRAPPER -->

        <!-- JAVASCRIP FILES-->
        <script src="..."></script>

    </body>
</html>

Installation guide

Please follow this step for installation guide for AngularJS version.

Step Description
1

Copy all files in folder angularjs and then put on root directory project   (path : /production/admin/angularjs)
2

Copy folder assets put on root directory project   (path : /assets)
3

Remove folder frontend on assets
4

Open file index.html, account.html on your root directory and then replace path resources, ex : ../../../assets to assets and then remove this line code ../../../production/admin/angularjs/
5

Open controllers.js, directives.js replace to ../../../assets to assets   (path : /production/admin/angularjs/js/controllers.js)   (path : /production/admin/angularjs/js/directives.js)
6

Open config.js and then search dataPath replace to production/admin/angularjs/data to data   (path : /production/admin/angularjs/js/config.js)
7

Open browser ex : http://localhost/yourproject/#/dashboard

Congratulations! you can now starting Blankon AngularJS app :D
