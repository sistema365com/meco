MixItUp is a jQuery plugin providing animated filtering and sorting. MixItUp is great for managing any categorized or ordered content like portfolios, galleries and blogs, but can also function as a powerful tool for engaging application UI and data-visualisation.

Official website : https://mixitup.kunkalabs.com
