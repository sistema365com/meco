Reveal Animations When You Scroll. Very Animate.css Friend :-) Easily customize animation settings: style, delay, length, offset, iterations...

Official website : http://mynameismatthieu.com/WOW/index.html

Basic Usage

  1. Include the stylesheet animate.css on your document's <head>

    <link rel="stylesheet" href="css/animate.css">

  2. Link and activate WOW.js

    <script src="js/wow.min.js"></script>
<script>
    new WOW().init();
</script>

  3. Reveal CSS Animations

    Make an element revealable

    <div class="wow">
    Content to Reveal Here
</div>

  4. Choose the animation style

    <div class="wow bounceInUp">
    Content to Reveal Here
</div>
