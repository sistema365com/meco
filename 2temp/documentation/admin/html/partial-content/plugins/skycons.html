Skycons is a set of ten animated weather glyphs, procedurally generated by JavaScript using the HTML5 canvas tag.

Official website : http://darkskyapp.github.io/skycons

SOURCE CODE (HTML)

<canvas id="clear-day" width="50" height="50"></canvas>

REQUIRED FILES (JS)

<script src="../../../assets/global/plugins/bower_components/skycons/skycons.js"></script>

SCRIPT (JS)

var icons = new Skycons({"color": "white"},{"resizeClear": true}),
                    list  = [
                        "clear-day", "clear-night", "partly-cloudy-day",
                        "partly-cloudy-night", "cloudy", "rain", "sleet", "snow", "wind",
                        "fog"
                    ],
                    i;

            for(i = list.length; i--; )
                icons.set(list[i], list[i]);

            icons.play();
