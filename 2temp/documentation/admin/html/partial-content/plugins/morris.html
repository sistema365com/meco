morris.js good-looking charts shouldn't be difficult

Official website : http://morrisjs.github.io/morris.js

Demo page : chart-morris.html

Available chart type on Blankon :

  1. Line
  2. Area
  3. Bar
  4. Donut

<!-- BAR CHART -->

<!-- DONUT CHART -->
<div id="morris-donut-chart"></div>
