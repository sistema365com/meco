C3 enables deeper integration of charts into your application.

Official website : http://c3js.org

Demo page : chart-c3js.html

Available chart type on Blankon :

  1. Line
  2. Bar
  3. Area
  4. Stacked
  5. Pie
  6. Donut
  7. Combination

SOURCE CODE (HTML)

<!-- LINE CHART -->
<div id="c3js-line-chart"></div>

<!-- BAR CHART -->
<div id="c3js-bar-chart"></div>

<!-- AREA CHART -->
<div id="c3js-area-chart"></div>

<!-- STACKED CHART -->
<div id="c3js-stacked-chart"></div>

<!-- PIE CHART -->
<div id="c3js-pie-chart"></div>

<!-- DONUT CHART -->
<div id="c3js-donut-chart"></div>

<!-- COMBINATION CHART -->
<div id="c3js-combination-chart"></div>
