Basic Usage

  1. Include the stylesheet on your document's <head>

    <head>
    <link href="../../../../assets/global/plugins/bower_components/spinkit/css/spinners/1-rotating-plane.css" rel="stylesheet">
</head>

  2. Add this code into body tag HTML.

    <div id="loader-wrapper">
    <div class="loader-brand">
        <div class="sk-rotating-plane"></div>
        <div class="loader-text">
            APP NAME
        </div>
    </div>
    <div class="loader-section section-left"></div>
    <div class="loader-section section-right"></div>
</div>

  3. Add code tyle like this, you can use SASS or CSS. We provide 2 code style.

    SASS 

    /* ========================================================================
 * PRELOADER
 * ======================================================================== */
#loader-wrapper {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  z-index: 10001;
  .loader-section {
    position: fixed;
    top: 0;
    width: 51%;
    height: 100%;
    z-index: 1000;
    &.section-left {
      left: 0;
    }
    &.section-right {
      right: 0;
    }
  }
}

.loaded {
  #loader-wrapper {
    visibility: hidden;
    .loader-section {
      &.section-left {
        -webkit-transform: translateX(-100%);  /* Chrome, Opera 15+, Safari 3.1+ */
        -ms-transform: translateX(-100%);  /* IE 9 */
        transform: translateX(-100%);  /* Firefox 16+, IE 10+, Opera */

        -webkit-transition: all 0.7s 0.3s cubic-bezier(0.645, 0.045, 0.355, 1.000);  /* Android 2.1+, Chrome 1-25, iOS 3.2-6.1, Safari 3.2-6  */
        transition: all 0.7s 0.3s cubic-bezier(0.645, 0.045, 0.355, 1.000);  /* Chrome 26, Firefox 16+, iOS 7+, IE 10+, Opera, Safari 6.1+  */
      }
      &.section-right {
        -webkit-transform: translateX(100%);  /* Chrome, Opera 15+, Safari 3.1+ */
        -ms-transform: translateX(100%);  /* IE 9 */
        transform: translateX(100%);  /* Firefox 16+, IE 10+, Opera */

        -webkit-transition: all 0.7s 0.3s cubic-bezier(0.645, 0.045, 0.355, 1.000);  /* Android 2.1+, Chrome 1-25, iOS 3.2-6.1, Safari 3.2-6  */
        transition: all 0.7s 0.3s cubic-bezier(0.645, 0.045, 0.355, 1.000);  /* Chrome 26, Firefox 16+, iOS 7+, IE 10+, Opera, Safari 6.1+  */
      }
    }
  }
}

.loader-brand{
  z-index: 99999;
  display: block;
  position: relative;
  top: 40%;
  width: 72px;
  height: 66px;
  margin: 0 auto;
}

.loader-text{
  color: #FFF;
  letter-spacing: 4px;
  font-size: 14px;
}

.sk-rotating-plane{
  background-color: #FFFFFF;
  margin: 20px auto;
}

  4. In scripts section directly bellow jQuery link include following javascript code:

    <!-- JQuery -->
<script type="text/javascript" src="../../../../assets/global/plugins/bower_components/jquery/dist/jquery.min.js"></script>

<!-- Preloader -->
<script type="text/javascript">
    $(document).ready(function () {
        setTimeout(function(){
            $('body').addClass('loaded');
        }, 3000);
    });
</script>
