Base HTML structure

This theme is a fluid layout with two columns and or three columns. All of the information within the page content area is nested within a div with id of "page-content". The sidebar left's (column #2) content is within a div with an id of "sidebar-left". The sidebar right's (column #3) content is within a div with an id of "sidebar-right". The general template structure is the same throughout the template. Here is the general structure.

Installation guide

Please follow this step for installation guide for PHP version

Step Description
1

Copy all files in folder php and then put on root directory project   (path : /production/admin/php)
2

Copy folder assets put on root directory project   (path : /assets)
3

Remove folder frontend on assets
4

Open file config.php and then change path getBase_url from http://localhost/blankon-fullpack-admin-theme/1.0.8/ to http://localhost/yourproject/   (path : /production/admin/php/config/config.php)
5

Open file config.php and then change path getMenu_url from production/admin/php/index.php to index.php   (path : /production/admin/php/config/config.php)
6

Open browser ex : http://localhost/yourproject/index.php
7

Congratulations! you can now starting Blankon PHP app :D
