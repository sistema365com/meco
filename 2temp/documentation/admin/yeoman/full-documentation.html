Getting started

Yeoman helps you to kickstart new projects, prescribing best practices and tools to help you stay productive. To do so, we provide a generator ecosystem. A generator is basically a plugin that can be run with the `yo` command to scaffold complete projects or useful parts.

The Grunt ecosystem is huge and it's growing every day. With literally hundreds of plugins to choose from, you can use Grunt to automate just about anything with a minimum of effort.

Bower works by fetching and installing packages from all over, taking care of hunting, finding, downloading, and saving the stuff you’re looking for. Web sites are made of lots of things — frameworks, libraries, assets, utilities, and rainbows. Bower manages all these things for you.

Base HTML structure

This theme is a fluid layout with two columns and or three columns. All of the information within the page content area is nested within a div with id of "page-content". The sidebar left's (column #2) content is within a div with an id of "sidebar-left". The sidebar right's (column #3) content is within a div with an id of "sidebar-right". The general template structure is the same throughout the template. Here is the general structure.

HTML structure

<!DOCTYPE html>
<html lang="en" data-ng-app="blankonApp">
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
        <title data-ng-bind="$state.current.data.pageTitle + ' | BLANKON Yeoman'"></title>
        <!-- CSS FILES-->
        <link data-ng-href="{{...}}" rel="stylesheet">
    </head>

    <body data-loading-animation>

        <!-- START WRAPPER -->
        <section id="wrapper">

            <!-- START HEADER -->
            <header data-ng-include="'partials/header.html'" id="header">
                ...
            </header><!-- /#header -->
            <!--/ END HEADER -->

            <!-- START SIDEBAR LEFT -->
            <aside data-sidebar-left-nicescroll data-sidebar-minimize data-ng-include="'partials/sidebar-left.html'" id="sidebar-left">
                ...
            </aside><!-- /#sidebar-left -->
            <!--/ END SIDEBAR LEFT -->

            <!-- START PAGE CONTENT -->
            <section id="page-content">

                    <!-- START HEADER CONTENT -->
                    <div class="header-content">
                            ...
                    </div><!-- /.header-content -->
                    <!--/ END HEADER CONTENT -->

                    <!-- START BODY CONTENT -->
                    <div data-ui-view class="body-content">
                            ...
                    </div><!-- /.body-content -->
                    <!--/ END BODY CONTENT -->

                    <!-- START FOOTER CONTENT -->
                    <div class="footer-content">
                            ...
                    </div><!-- /.footer-content -->
                    <!--/ END FOOTER CONTENT -->

            </section><!-- /#page-content -->
            <!--/ END PAGE CONTENT -->

            <!-- BEGIN SIDEBAR RIGHT -->
            <aside data-choose-themes data-navbar-color data-sidebar-color data-layout-setting data-ng-include=" 'partials/sidebar-right.html' " id="sidebar-right">
                ...
            </aside><!-- /#sidebar-right -->
            <!--/ END SIDEBAR RIGHT -->

        </section><!-- /#wrapper -->
        <!--/ END WRAPPER -->

        <!-- JAVASCRIP FILES-->
        <script src="..."></script>

    </body>
</html>

<!--[if !IE]><!--> <html lang="en" data-ng-app="blankonApp"> <!--<![endif]-->

Installation guide

Please follow this step for installation guide for Codeigniter version

Step Description
1

Go to official Yeoman website on here : http://yeoman.io
2

Set up your dev environment

URL : http://yeoman.io/codelab/setup.html
3

Install a Yeoman generator (yo angular)

URL : http://yeoman.io/codelab/install-generators.html

Open console and then type

  • 1. $mkdir yourproject Please give access root on this directory (ex : sudo chmod 777 -R yourproject)
  • 2. $yo angular Please select your framework, preprocessor etc.
  • 3. Now Yeoman scaffolding project is ready
4

Implement Blankon with yeoman

  • 1. Copy all files in folder yeoman and then put on root directory project   (path : /production/admin/yeoman)
  • 2. Copy folder assets put on root directory project   (path : /assets)
  • 3. Remove folder frontend on assets
  • 4. Copy files bower.json, Gruntfile.js, package.json replace on root directory project   (path : /production/admin/yeoman/bower.json)
  • 4. Copy folder app replace app folder on root directory project   (path : /production/admin/yeoman/app)
  • 5. Copy folder test replace test folder on root directory project   (path : /production/admin/yeoman/test)
5

Update dependency library
6

Open console and then goto your project just type $bower update
7

Running yeoman

Open console and then goto your project just type $grunt serve or $grunt serve --force
8

Wait the moment, Yeoman just running for open browser with this URL : http://localhost:9000/#/dashboard

Congratulations! you can now starting Blankon Yeoman app :D
