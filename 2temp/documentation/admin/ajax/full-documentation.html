Getting started

Ajax or short for asynchronous JavaScript + XML. is a group of interrelated Web development techniques used on the client-side to create asynchronous Web applications. With Ajax, Web applications can send data to, and retrieve data from, a server asynchronously (in the background) without interfering with the display and behavior of the existing page.

The best of all: it makes websites so fast!

File structure

Check the files structure below, you should get them all within template bundle AJAX version. You can see file on : production/admin/ajax

Folder & File Description
js Used to handle core AJAX script.
php Used to store all support PHP file.
views Used to store views file pages.
account Used to store account design page.
index Used to call first time blankon AJAX.

Base HTML structure

This theme is a fluid layout with two columns and or three columns. All of the information within the page content area is nested within a div with id of "page-content". The sidebar left's (column #2) content is within a div with an id of "sidebar-left". The sidebar right's (column #3) content is within a div with an id of "sidebar-right". The general template structure is the same throughout the template. Here is the general structure.

Installation guide

Step Description
1

Copy all files in folder ajax and then put on root directory project   (path : /production/admin/ajax)
2

Copy folder assets put on root directory project   (path : /assets)
3

Remove folder frontend on assets
4

Open all files html on your root directory and then replace path resources, ex : ../../../assets to assets
5

Open apps.js then search globalPluginsPath change to assets/global/plugins/bower_components and globalDataPath replace to ../../../assets to assets   (path : /assets/admin/js/apps.js)

Open browser ex : http://localhost/yourproject/#dashboard
