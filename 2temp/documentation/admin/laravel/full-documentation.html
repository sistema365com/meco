Base HTML structure

This theme is a fluid layout with two columns and or three columns. All of the information within the page content area is nested within a div with id of "page-content". The sidebar left's (column #2) content is within a div with an id of "sidebar-left". The sidebar right's (column #3) content is within a div with an id of "sidebar-right". The general template structure is the same throughout the template. Here is the general structure.

Installation guide

This manual explains how to installation Blankon Laravel on your project using Laravel Framework.

Step Description
1

Install laravel application via composer

composer create-project laravel/laravel laravel5 5.0 —prefer-diet
2

Copy and merge app, config and resources folders from production/admin to /laravel5
3

Copy merge Blankon assets folder to /laravel5/public
4

Go to config folder, open constant.php

vim /laravel5/config/constant.php , ‘assetUrl’ => ‘http://localhost:8000/assets
5

Type on terminal

php artisan serve
6

Open browser

Example URL browser : http://localhost:8000

Congratulations! you can now starting Blankon Laravel app :D

Composer support

Blankon's Laravel version using Composer as dependency manager, If you do not understand Composer, you can visit the official website Composer www.getcomposer.org

Here we will explain how to use bower quickly and easily.

Step Description
1

Open console and go to laravel folder on blankon

/production/admin/laravel
2

Install Composer

curl -sS https://getcomposer.org/installer | php
3

Install dependency

php composer.phar install
4

Testing Laravel on browser

http://localhost/sampleApp/public

If your operating system is Windows, download and run Composer-Setup.exe - it will install the latest composer version whenever it is executed.

Now we can use laravel in our development, Its easy. :D
